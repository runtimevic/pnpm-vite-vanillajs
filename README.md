 ### Create project:

 - ```pnpm create vite```
 ***
 ### Run development project:
 
 - ```cd pnpm-vite-vanillajs```

 - ```pnpm install```  

 - ```pnpm run dev ```
***
### Build, create dist:

 - ```pnpm run build ```

 - ```copy, paste folder dist rename docs ```
